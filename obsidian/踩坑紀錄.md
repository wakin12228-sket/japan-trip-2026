## GitHub Pages 404 / 沒有 /src 選項

**日期：** 2026-06-20  
**嚴重度：** 中

**現象：**  
Pages 設定只有 `/(root)` 和 `/docs`；選 root 或尚未部署時開網址顯示 404。

**原因：**  
GitHub Pages（branch 部署）不支援 `/src`；Private repo 免費方案無法開 Pages。

**解法：**  
1. Repo 改 Public  
2. 網站檔放 `docs/`，Pages 選 `main` + `/docs`  
3. push 後等 1–3 分鐘

**預防：**  
靜態站若用 GitHub Pages，一開始就把可部署檔放在 `docs/`。
